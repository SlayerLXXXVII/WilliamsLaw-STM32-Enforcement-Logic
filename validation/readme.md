# Validation Data for Williams Law

This folder contains datasets and plots validating the collapse model.

## Files

- `entropy_threshold.csv`  
  Columns: Time_ps, MicrostateCount, EntropyRate_J_per_K_s

- `collapse_plot.png`  
  Plot of Φ vs Time and dΦ/dt vs Sₜ overlay.

- `film_test_results.csv`  
  Columns: DeltaT_C, Gradient_K_per_mm, PhiThreshold, CollapseTime_s, ErrorPercent
